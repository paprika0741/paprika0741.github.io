基本问题与主要困难

  • 基本问题
    • 形态学(morphology)：词由有意义的基本单位—词素（词根、前缀、后缀、词尾）的构成问题、单词识别/汉语分词问题
    • 句法(syntax)问题：研究句子结构成分之间的相互关系和组成句子序列的规则（主谓宾）
    • 语义(semantic):研究如何从一个语句中推导出词的意义，以及这些词在语句句法结构中的作用来推导出该语句的意义（同一个词在不同语境下会有不同意思）
    • 语用学(pragmatics):研究在不同上下文语句的应用，以及上下文对语句理解产生的影响
  • 主要困难
    • Ambiguity
      • Lexical ambiguity 词法歧义
      • Part-of-speech ambiguity 词性歧义
      • Structural ambiguity 结构歧义

        主语 谓语 状语 宾语修饰关系
  • unknown language phenomena
    大量未知语言现象：新词、人名、地名、术语、新含义、新句法、新句型

  • Question Answering & Dialogue Systems 问答系统

朴素贝叶斯法

  • 基于贝叶斯定理和特征条件独立假设的分类方法

  • 训练数据集 $T=\left\{\left(x_{1}, y_{1}\right),\left(x_{2}, y_{2}\right), \cdots,\left(x_{N}, y_{N}\right)\right\}$

  • 训练时
    通过训练集学习以下先验概率分布和条件概率分布

    先验概率分布：
$P\left(Y=c_{k}\right), \quad k=1,2, \cdots, K$
条件概率分布：
$P\left(X=x \mid Y=c_{k}\right)=P\left(X^{(1)}=x^{(1)}, \cdots, X^{(n)}=x^{(n)} \mid Y=c_{k}\right), \quad k=1,2, \cdots, K$

    ​ 又由条件独立性假设

    ​ 用于分类的特征在类确定的条件下都是条件独立的
    ​ $\begin{aligned} P\left(X=x \mid Y=c_{k}\right) &=P\left(X^{(1)}=x^{(1)}, \cdots, X^{(n)}=x^{(n)} \mid Y=c_{k}\right) \\ &=\prod_{j=1}^{n} P\left(X^{(j)}=x^{(j)} \mid Y=c_{k}\right) \end{aligned}$

  • 分类时
    输入x，通过学习到的模型计算后验概率分布$P(Y=c_k|X=x)$,将后验概率最大的类作为x的类输出

    $P\left(Y=c_{k} \mid X=x\right)=\frac{P\left(X=x \mid Y=c_{k}\right) P\left(Y=c_{k}\right)}{\sum_{k} P\left(X=x \mid Y=c_{k}\right) P\left(Y=c_{k}\right)}$

    代入
    $P\left(Y=c_{k} \mid X=x\right)=\frac{P\left(Y=c_{k}\right) \prod_{j} P\left(X^{(j)}=x^{(j)} \mid Y=c_{k}\right)}{\sum_{k} P\left(Y=c_{k}\right) \prod_{j} P\left(X^{(j)}=x^{(j)} \mid Y=c_{k}\right)}, \quad k=1,2, \cdots, K$

    于是，朴素贝叶斯分类器可以表示为
    $y=f(x)=\arg \max _{c_{k}} \frac{P\left(Y=c_{k}\right) \prod_{j} P\left(X^{(j)}=x^{(j)} \mid Y=c_{k}\right)}{\sum_{k} P\left(Y=c_{k}\right) \prod_{j} P\left(X^{(j)}=x^{(j)} \mid Y=c_{k}\right)}$

​ 注意到，对分母所有$c_k$都是相同的

​ $y=\arg \max _{c_{k}} P\left(Y=c_{k}\right) \prod_{j} P\left(X^{(J)}=x^{(j)} \mid Y=c_{k}\right)$

最大化后验概率P（d）
可以看成一个常数
P（C）先验概率
How often does this class occur，How often does this class occur

$P\left(x_{1}, x_{2}, \cdots, x_{n} \mid c\right)$ 独立假设 👇

$P\left(x_{1}, \ldots, x_{n} \mid c\right)=P\left(x_{1} \mid c\right) P\left(x_{2} \mid c\right) P\left(x_{3} \mid c\right) \ldots P\left(x_{n} \mid c\right)$

$c_{M A P}=\underset{c \in C}{\operatorname{argmax}} P\left(x_{1}, x_{2}, \cdots, x_{n} \mid c\right) P(c)$
$c_{N B}=\underset{c \in C}{\operatorname{argmax}} P\left(c_{j}\right) \prod_{x, \in X} P\left(x_{i} \mid c\right)$

Problem：floating-point underflow
Solution：对数空间

$c_{\mathrm{NB}}=\underset{c_{j} \in C}{\operatorname{argmax}}\left[\log P\left(c_{j}\right)+\sum_{i \in \text { positions }} \log P\left(x_{i} \mid c_{j}\right)\right]$

$\Rightarrow$ linear model ， linear classifier

Learning

  • 计算先验部分
    统计每个类别的概率
    每个类别每个词语的条件概率
    $\hat{P}\left(c_{j}\right)=\frac{N_{c_{j}}}{N_{\text {total }}}$
    $\hat{P}\left(w_{i} \mid c_{j}\right)=\frac{\operatorname{count}\left(w_{i}, c_{j}\right)}{\sum_{w\llcorner} \operatorname{count}\left(w, c_{j}\right)}$

    具体实现：同一类别拼接成一个文档，统计词的概率

  • Problem：训练集从未出现过 出现在测试集
    Laplace (add-1) smoothing

    $\begin{aligned} \hat{P}\left(w_{i} \mid c\right) &=\frac{\operatorname{count}\left(w_{i}, c\right)+1}{\sum_{w\lfloor V}(\operatorname{count}(w, c)+1)} \\ &=\frac{\operatorname{count}\left(w_{i}, c\right)+1}{\left(\sum_{w \in V} \operatorname{count}(w, c)\right)+|V|} \end{aligned}$

Cosine for computing word similarity 计算词语的相似度

Dot product

$\operatorname{dot} \operatorname{product}(\mathbf{v}, \mathbf{w})=\mathbf{v} \cdot \mathbf{w}=\sum_{i=1}^{N} v_{i} w_{i}=v_{1} w_{1}+v_{2} w_{2}+\ldots+v_{N} w_{N}$

当两个向量在相同维度上具有较大值时，点积往往较高

$\Rightarrow $ 向量之间有用的相似性度量

Problem:

  • Dot product favors long vectors
    长度的影响,如果向量较长，则点积更高
  • Dot product overly favors frequent words
    有些频繁使用的词有很长的向量（因为它们与其他单词一起出现了很多次）

Cosine

normalized dot product

$\operatorname{cosine}(\vec{v}, \vec{w})=\frac{\vec{v} \cdot \vec{w}}{|\vec{v}||\vec{w}|}=\frac{\sum_{i=1}^{N} v_{i} w_{i}}{\sqrt{\sum_{i=1}^{N} v_{i}^{2}} \sqrt{\sum_{i=1}^{N} w_{i}^{2}}}$

  • -1 vectors point in opposite directions
  • +1 vectors point in same directions
  • 0 vectors are orthogonal

频数不可能出现负的,所以范围0 - 1

TF-IDF

上述都是基于频率 词语出现次数/在文档中出现次数

问题： the it they… 无用的词语频率很高
tf-idf：为了使有用词语频率 ↑ 无用 ↓

  • tf 词频 term frequency

    ​ $\mathrm{tf}_{t, d}=\operatorname{count}(t, d)$

    ​ $\mathrm{tf}_{t, d}=\log _{10}(\operatorname{count}(t, d)+1)$

  • idf 词语在很多文档出现 重要性降低

    ​ “the” “it” —> low idf

    • Document Frequency (df)
      在多少个文档里面出现过
      $\operatorname{idf}_{t}=\log _{10}\left(\frac{N}{\mathrm{df}_{t}}\right)$ N是集合中的文档总数。

  • tf-idf value for word $t$ in document $d$ :

对于tf-idf 文档可以是任何内容，经常把每一段称为文档

PMI: (Pointwise mutual information)

  • 计算两个词语在概率上的分布

  • Pointwise mutual information 点式互信息：

    事件x和y共同发生的次数和独立发生的次数

    $\operatorname{PMI}(X, Y)=\log _{2} \frac{P(x, y)}{P(x) P(y)}$

    $\operatorname{PMI}\left(\right.$ word $_{1}$, word $\left._{2}\right)=\log _{2} \frac{P\left(\text { word }_{1}, \text { word }_{2}\right)}{P\left(\text { word }_{1}\right) P\left(\text { word }_{2}\right)}$

    范围 $(-\infty, +\infty)$

  • 如果完全独立 log = 0，负的结果没有意义，如果<0 强制设为零

  • PPMI(word $_{1}$, word $\left._{2}\right)=\max \left(\log _{2} \frac{P\left(\text { word }_{1}, \text { word }_{2}\right)}{P\left(\text { word }_{1}\right) P\left(\text { word }_{2}\right)}, 0\right)$

    Positive PMI

为什么dot product 而不是cosine

  • Cosine就是normalized dot product，这里词向量的维度都是一样的,不存在长度的差别，

  • 优化的时候用随机梯度下降，cos 不容易计算梯度，用点积很好优化

sigmoid ：将相似度转换成概率

相似度高，则意味着+的概率大

如何计算P(+|w,c)

窗口里面其他的词语 全部乘起来（假设独立）

​ $\begin{aligned} P\left(+\mid w, c_{1: L}\right) &=\prod_{i=1}^{L} \sigma\left(c_{i} \cdot w\right) \\ \log P\left(+\mid w, c_{1: L}\right) &=\sum_{i=1}^{L} \log \sigma\left(c_{i} \cdot w\right) \end{aligned}$

通过计算 w (embeddings) 和 c1:L(embeddings)的相似性，计算w出现在窗口的概率
