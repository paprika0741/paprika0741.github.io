Part of Speech and Named Entities

词性 part of speech

用词性标记在句子中注释每个单词 （label）

  • Closed class 封闭类

​ 固定的

  • Open class 开放类 类别词语里面的词可能一直在变化

    ​ Nouns (Googler), Verbs (Google), Adjectives (geeky), Abverb (automagically)

ADP: adposition

词性标注任务 Part of Speech Tagging

useful for other NLP tasks

  • parse 从语法上描述或分析（词句等） ：POS标记可以改进语法解析

  • MT(机器翻译)： 有些需要调整名词和形容词顺序 （例如西班牙语->英语）

  • Sentiment or affective tasks ：感情或情感任务中词性很重要，评价的是名词，描述是形容词

  • Text-to-speech 文本和语音 ：词性不一样 发音不同

Difficulty

  • Roughly 15% of word types are ambiguous 大约15%的单词词性是不明确的

  • Ambiguity

    一个词可以有多个 词性

实现：

  • 提取句子(不需要考虑不同句子的关系，消除标点符号的歧义，检查句子边界)

  • 句子中每个词语的词性确认

​

  • 可用信息：

    • 先验知识

      • 词表中有词语的词性类别， 去除大部分干扰

        比如 will 通常是 AUX

      • 相邻单词的标识
        the 意味着下一个词不会是动词

    • Morphology and wordshape: 形态学和文字形态

      词法 Morphology and 词形wordshape:
      前缀 后缀 大小写

Standard algorithms for POS tagging

  • Supervised Machine Learning Algorithms

    • Hidden Markov Models 隐马尔可夫

    • Conditional Random Fields (CRF) 条件随机场

    • Neural sequence models (RNNs or Transformers)

    • Large Language Models (like BERT), finetuned

      All required a hand-labeled training set, all about equal performance

      (97% on English) 需要一个手工标记的训练集 / 等性能

    • 利用先验信息：
      HMMs and CRFs ：Via human created features
      Neural LMs ：Via representation learning

问题：

  • 标准分类问题假设单个案例是独立的(i.i.d.：独立且同分布）

  • Many NLP problems do not satisfy this assumption and involve making many connected decisions, each resolving a different ambiguity, but which are mutually dependent 相互依赖的

  • 深度学习 联合考虑

    Sequence Labeling Problem 序列标记问题

命名实体识别 Named Entity Recognition

PER (Person) 人名：

LOC (Location): 地名：

ORG (Organization): 组织

GPE (Geo-Political Entity):地源政治

dates日期 times 时间 prices价格

特点：

  • 长度多于一个，几个词语组成
  • 词语拆分可能有不同的意思，需要整体提取出来，整体识别
    比如北京大学

任务：

named entity recognition (NER):

  • 找出一个片段 （命名实体）对应的位置

  • 找出类别，标记属于哪一种命名实体

Why NER？

Sentiment analysis 情绪分析：消费者对特定公司还是个人的情绪？

Question Answering 问答系统：问题里面出现问答实体，回答有关一个实体的问题，需要将实体完整地识别出来

Information Extraction：Extracting facts about entities from text

Why NER is hard

  • Segmentation
    分隔的问题：NER需要识别边界，要找候选的边界
    词性标注每个词语都是识别的单元 不存在边界的问题

  • Type ambiguity
    歧义

基于序列标注

BIO Tagging

句子里面每一个词语打一个标记，每个位置对应 B I O中的一个

B: token that begins a span

I: tokens inside a span

O: tokens outside of any span

  • 考虑实体的类别 BIOES

  • IO

然后根据标记合起来

左边 B 右边 I 合并

O 跳过

Standard algorithms for NER

Supervised Machine Learning

  • Hidden Markov Models
  • Conditional Random Fields (CRF)
  • Neural sequence models (RNNs or Transformers)
  • Large Language Models (like BERT), finetuned
