朴素贝叶斯法

  • 基于贝叶斯定理和特征条件独立假设的分类方法

  • 训练数据集 $T=\left\{\left(x_{1}, y_{1}\right),\left(x_{2}, y_{2}\right), \cdots,\left(x_{N}, y_{N}\right)\right\}$

  • 训练时
    通过训练集学习以下先验概率分布和条件概率分布

    先验概率分布：
$P\left(Y=c_{k}\right), \quad k=1,2, \cdots, K$
条件概率分布：
$P\left(X=x \mid Y=c_{k}\right)=P\left(X^{(1)}=x^{(1)}, \cdots, X^{(n)}=x^{(n)} \mid Y=c_{k}\right), \quad k=1,2, \cdots, K$

    ​ 又由条件独立性假设

    ​ 用于分类的特征在类确定的条件下都是条件独立的
    ​ $\begin{aligned} P\left(X=x \mid Y=c_{k}\right) &=P\left(X^{(1)}=x^{(1)}, \cdots, X^{(n)}=x^{(n)} \mid Y=c_{k}\right) \\ &=\prod_{j=1}^{n} P\left(X^{(j)}=x^{(j)} \mid Y=c_{k}\right) \end{aligned}$

  • 分类时
    输入x，通过学习到的模型计算后验概率分布$P(Y=c_k|X=x)$,将后验概率最大的类作为x的类输出

    $P\left(Y=c_{k} \mid X=x\right)=\frac{P\left(X=x \mid Y=c_{k}\right) P\left(Y=c_{k}\right)}{\sum_{k} P\left(X=x \mid Y=c_{k}\right) P\left(Y=c_{k}\right)}$

    代入
    $P\left(Y=c_{k} \mid X=x\right)=\frac{P\left(Y=c_{k}\right) \prod_{j} P\left(X^{(j)}=x^{(j)} \mid Y=c_{k}\right)}{\sum_{k} P\left(Y=c_{k}\right) \prod_{j} P\left(X^{(j)}=x^{(j)} \mid Y=c_{k}\right)}, \quad k=1,2, \cdots, K$

    于是，朴素贝叶斯分类器可以表示为
    $y=f(x)=\arg \max _{c_{k}} \frac{P\left(Y=c_{k}\right) \prod_{j} P\left(X^{(j)}=x^{(j)} \mid Y=c_{k}\right)}{\sum_{k} P\left(Y=c_{k}\right) \prod_{j} P\left(X^{(j)}=x^{(j)} \mid Y=c_{k}\right)}$

​ 注意到，对分母所有$c_k$都是相同的

​ $y=\arg \max _{c_{k}} P\left(Y=c_{k}\right) \prod_{j} P\left(X^{(J)}=x^{(j)} \mid Y=c_{k}\right)$

Naive Bayes Classifier

  • Based on Bayes rule
    假设词语之间是独立的

  • Bag of words

    预处理 词语标准化 词根还原
    统计每个词出现的次数

  • d: document
    c: class
    $P(c \mid d)=\frac{P(d \mid c) P(c)}{P(d)}$

忽略词语之间的顺序关系

最大化后验概率P（d）
可以看成一个常数
P（C）先验概率
How often does this class occur，How often does this class occur

$P\left(x_{1}, x_{2}, \cdots, x_{n} \mid c\right)$ 独立假设 👇

$P\left(x_{1}, \ldots, x_{n} \mid c\right)=P\left(x_{1} \mid c\right) P\left(x_{2} \mid c\right) P\left(x_{3} \mid c\right) \ldots P\left(x_{n} \mid c\right)$

$c_{M A P}=\underset{c \in C}{\operatorname{argmax}} P\left(x_{1}, x_{2}, \cdots, x_{n} \mid c\right) P(c)$
$c_{N B}=\underset{c \in C}{\operatorname{argmax}} P\left(c_{j}\right) \prod_{x, \in X} P\left(x_{i} \mid c\right)$

Problem：floating-point underflow
Solution：对数空间

$c_{\mathrm{NB}}=\underset{c_{j} \in C}{\operatorname{argmax}}\left[\log P\left(c_{j}\right)+\sum_{i \in \text { positions }} \log P\left(x_{i} \mid c_{j}\right)\right]$

$\Rightarrow$ linear model ， linear classifier

Learning

  • 计算先验部分
    统计每个类别的概率
    每个类别每个词语的条件概率
    $\hat{P}\left(c_{j}\right)=\frac{N_{c_{j}}}{N_{\text {total }}}$
    $\hat{P}\left(w_{i} \mid c_{j}\right)=\frac{\operatorname{count}\left(w_{i}, c_{j}\right)}{\sum_{w\llcorner} \operatorname{count}\left(w, c_{j}\right)}$

    具体实现：同一类别拼接成一个文档，统计词的概率

  • Problem：训练集从未出现过 出现在测试集
    Laplace (add-1) smoothing

    $\begin{aligned} \hat{P}\left(w_{i} \mid c\right) &=\frac{\operatorname{count}\left(w_{i}, c\right)+1}{\sum_{w\lfloor V}(\operatorname{count}(w, c)+1)} \\ &=\frac{\operatorname{count}\left(w_{i}, c\right)+1}{\left(\sum_{w \in V} \operatorname{count}(w, c)\right)+|V|} \end{aligned}$

Precision, Recall, and F measure

评价分类器

混淆矩阵

  • true代表预测和实际一样
    false 表示预测和实际不一样
  • true positive真阳：
    positive 表示预测为positive

  • false positive

    预测是错的
    negative 表示预测为negative

  • recall 差全率 召回率
  • accurary准确率：$\frac{true}{true+false}$
  • precision :$\frac{true \ positive}{true \ positive \ + false \ positive}$

Why don’t we use accuracy as our metric?

如果只用accurary，假如100万样本，positive只有100个，不需要学习，全部分为negative..准确度高

F measure

a single number that combines P and R
$F_{\beta}=\frac{\left(\beta^{2}+1\right) P R}{\beta^{2} P+R}$

balanced $F_1$ ($\beta = 1$)
$F_{1}=\frac{2 P R}{P+R}$
