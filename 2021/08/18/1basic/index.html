Markov Property

The history of states: $h_t = {s_1,s_2,s_3, …,s_t}$

State $s_t$is Markovian if and only if:

$p(s_{t+1}|s_t) =p(s_{t+1}|h_t)$

$p(s_{t+1}|s_t,a_t) =p(s_{t+1}|h_t,a_t)$

The future is independent of the past given the present

Markov Chain + reward

Definition:

  • S is a (finite) set of states (s ∈ S)
  • P is dynamics/transition model that specifies $P(S_{t+1}= s_0|s_t = s)$

If finite number of states, R can be a vector

Why Discount Factor

  • Avoids infinite returns in cyclic Markov processes

  • Uncertainty about the future may not be fully represented

  • If the reward is financial, immediate rewards may earn more interest

    than delayed rewards

  • Animal/human behaviour shows preference for immediate reward

  • γ = 0: Only care about the immediate reward

    γ = 1: Future reward is equal to the immediate reward

Markov Decision Process (MDP)

Markov Reward Process with decisions

Definition

  • S is a finite set of states
  • A is a finite set of actions
  • $P^a$ is dynamics/transition model for each action
    $P(s_{t+1} = s’|s_t = s ,a_t = a) $
  • R is a reward function
    $R(s_t = s,a_t = a) = E[r_t|s_t= s,a_t= a]$
  • Discount factor γ ∈ [0, 1]

MDP is a tuple: (S, A,P,R, γ)

Policy in MDP

Policy specifies what action to take in each state
Give a state, specify a distribution over actions
Policy: $π(a|s) =P(a_t = a|s_t = s)$

Policies are stationary (time-independent),

$A_t ∼ π(a|s) for \ any\ t> 0$

Value function for MDP

Bellman方程

$q_{\pi}(s,a) = \mathbb{E}_{\pi}(G_t|S_t=s, A_t=a) = \mathbb{E}_{\pi}(R_{t+1} + \gamma R_{t+2} + \gamma^2R_{t+3}+…|S_t=s,A_t=a)$

$\begin{align} v_{\pi}(s) &= \mathbb{E}_{\pi}(R_{t+1} + \gamma R_{t+2} + \gamma^2R_{t+3}+…|S_t=s) \\ &= \mathbb{E}_{\pi}(R_{t+1} + \gamma (R_{t+2} + \gamma R_{t+3}+…)|S_t=s) \\ &= \mathbb{E}_{\pi}(R_{t+1} + \gamma G_{t+1} | S_t=s) \\ &= \mathbb{E}_{\pi}(R_{t+1} + \gamma v_{\pi}(S_{t+1}) | S_t=s) \end{align}$

也就是说，在t时刻的状态S_t和t+1时刻的状态S_{t+1}是满足递推关系的，即：

$v_{\pi}(s) = \mathbb{E}_{\pi}(R_{t+1} + \gamma v_{\pi}(S_{t+1}) | S_t=s)$

同样的方法，我们可以得到动作价值函数q_{\pi}(s,a)的贝尔曼方程：

$q_{\pi}(s,a) = \mathbb{E}_{\pi}(R_{t+1} + \gamma q_{\pi}(S_{t+1},A_{t+1}) | S_t=s, A_t=a)$

Bellman Expectation Equation

​

​ $v_{\pi}(s) = \sum\limits_{a \in A} \pi(a|s)q_{\pi}(s,a)$

状态价值函数是所有动作价值函数基于策略$\pi$的期望。通俗说就是某状态下所有状态动作价值乘以该动作出现的概率，最后求和，就得到了对应的状态价值。

$q_{\pi}(s,a) = R_s^a + \gamma \sum\limits_{s’ \in S}P_{ss’}^av_{\pi}(s’)$

Difference between Policy Iteration and Value Iteration

  • Policy iteration includes: policy evaluation + policy improvement, and the two are repeated iteratively until policy converges.

  • Value iteration includes: fifinding optimal value function + one

    policy extraction. There is no repeat of the two because once the value function is optimal, then the policy out of it should also be optimal (i.e. converged).
