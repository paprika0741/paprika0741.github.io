强化学习：多智能体强化学习基础

发表于 2021-08-21 分类于 RL 阅读次数：
本文字数： 5.1k 阅读时长 ≈ 5 分钟

Terminologies

There are n agents.
$\mathcal{S}$: state
$\mathcal{A^i}$: i-th agent’s action
State transition:
$p(s\prime|s,a^1,\dots,a^n)=\boldsymbol{P}(S’=s’|S=s,A^1=a^1,\dots,A^n=a^n)$
The next state S’ depends on all the agents’ actions.

Reward

$\mathcal{R^i}$: i-th agent’s reward
$R^i$ depends on $A^i$ as well as other agents’ actions ${A^j}_{j\neq i}$

Return

$R_t^i$ :the reward received by the i-th agent at time t
$U_t^i = R_t^i + R_{t+1}^i + R_{t+2}^i +R_{t+３}^i　+ \dots$

Discounted return:
$U_t^i = R_t^i + \gamma \cdot R_{t+1}^i +\gamma^2 \cdot R_{t+2}^i + \gamma^3 \cdot R_{t+３}^i　+ \dots$
$\gamma$ is the discount rate

Uncertainty in the Return

$R^i_t$ depends on $S_t$ and $A_t^1,A^2_t,\dots,A^n_t$

  • Uncertainty in $S_t$ is from the state transition, $p$

  • Uncertainty in $A_t^i$ is from the policy network, $\pi(a^i|s;\theta^i)$

  • $U_t^i=\sum^ \infty_{k=0} \gamma^k \cdot R^i_{t+k}$ depends on all the future states and actions

Policy network

  • Each agent has its own policy network: $\pi(a^i|s;\theta^i)$

  • Policy networks can be exchangeable:$\theta^i=\theta^j$

    Self-driving cars can have the same policy

  • Policy networks can be nonexchangeable
    $\theta^i \neq \theta^j$
    Soccer players have different roles..striker, defender, goalkeeper.

State-Value Function

$V^{i}\left(s_{t} ; \boldsymbol{\theta}^{1}, \cdots, \boldsymbol{\theta}^{n}\right)=\mathbb{E}\left[U_{t}^{i} \mid S_{t}=s_{t}\right]$

期望关于所有future actions and states except $S_t$

Randomness in actions:$A_{t}^{j} \sim \pi\left(\cdot \mid s_{t} ; \boldsymbol{\theta}^{j}\right), \text { for all } j=1, \cdots, n$

Settings

Fully cooperative

Agents collaborate to optimize a common return.
所有agents奖励相等 $R^1=R^2=\dots=R^n$

Fully competitive

One agent’s gain is the other agent’s loss.
比较典型的零和博弈，双方获得奖励总和为0

机器人格斗（零和博弈）/捕猎

$R^1 \propto -R^2 $

Mixed Cooperative & competitive

There are both cooperative setting and competitive setting

队伍之间是竞争关系，队伍内部是合作关系
比如星际争霸游戏

Self-interested

Agents are self-interested. Their rewards may or may not conflict
利己主义，比如股票自动交易系统/

Convergence

无法通过改进策略得到更大的期望汇报

Single agent:

  • Policy network: $\pi(a |s;\theta )$
  • State-value function: $V(s;\theta )$
  • $J(\boldsymbol{\theta})=\mathbb{E}_{S}[V(S ; \boldsymbol{\theta})]$ evaluate the policy
    (期望消掉s)
  • Learn the policy network’s parameter $\theta$ by
    $ \underset{\boldsymbol{\theta}} \max J\ (\boldsymbol{\theta})$
  • Convergence: $J(\boldsymbol{\theta})$ stops increasing

Multi-Agent

Nash Equilibrium：

While all the other agents’ policy remain the
same, the i-th agent cannot get better expected return by changing its own policy。其他agent不改变策略的情况下，一个agent改变策略无法让自己得到更多的回报。

Every agent is playing a best-response to the other agents’ policies. 每个agent都以最优的方式应对
Nash equilibrium indicates convergence because no one has any incentive to deviate.
纳什均衡情况下，所有agent都不会改变策略

Difficulty of MARL

如果直接套用single agent到MARL
n个agent，独立和环境交互，每个agent都能观测到环境s和奖励r

agent之间都不通信，不知道其他agent的信息

  • Policy network: $\pi(a^i |s;\theta^i )$

  • State-value function: $V^i(s;\theta^1,\dots,\theta^n )$

  • $J^i(\theta^1,\dots,\theta^n)=\mathbb{E}_{S}[V^i(S ; {\theta^1,\dots,\theta^n})]$ evaluate the policy

  • Learn the policy network’s parameter $\theta^i$ by
    $ \underset{\boldsymbol{\theta}^i} \max J^i\ (\boldsymbol{\theta})^i$

    每个agent都有自己的目标函数，各不相同，agent没有共同的目标，各自更新各自的参数$\theta^i$,

    It may not converge…
    一个agent更新策略，其他的目标函数都会发生变化

通信方式：Centralized VS Decentralized

Partial Observations

  • An agent may or may not have full knowledge of the state, s
  • $o^i$ ： the i-th agent’s observation
  • Partial observation $o^{i} \neq s$
  • Full observation $o^{1}=\cdots=o^{n}=S $

Architectures

Fully decentralized:

Every agent uses its own observations and rewards to learn its policy.
Agents do not communicate.彼此不交流

​

训练和执行都是独立的，本质是single agent RL

Fully Decentralized Actor-Critic Method

​

忽视了agent之间的相互影响，不应该是独立的

Fully centralized:

The agents send everything to the central controller.
The controller makes decisions for all the agents

Centralized Training & Centralized Execution

  • 中央控制器收集所有agent的观测 奖励 动作
    训练得到n个策略网络，$\pi\left(a^{i} \mid O^{1}, \cdots, o^{n} ; \boldsymbol{\theta}^{i}\right), \text { for all } i=1,2, \cdots, n$
    中央控制器n个策略网络，结构可以相同，参数不同
  • agent上没有策略网络，agent不做决策（决策需要所有的o，一个agent没有足够的信息），执行中央的指令

​

Centralized Actor-Critic Method
  • $\mathbf{a}=\left[a^{1}, a^{2}, \cdots, a^{n}\right]$ contain all the agents’ actions
  • $\mathbf{o}=\left[o^{1}, o^{2}, \cdots, o^{n}\right]$ contain all the agents’ observations.
  • The central controller knows $\mathbf{a} $ $\mathbf{o} $ and all the rewards
  • The controller has 8 policy networks and 8 value networks
    • Policy network (actor) for the i-th agent $\pi\left(a^{i} \mid \mathbf{o} ; \boldsymbol{\theta}^{i}\right) $ — policy gradient
    • Value network (critic) for the i-th agent
      $q\left(\mathbf{o}, \mathbf{a} ; \mathbf{w}^{i}\right)$ — TD algorithm
Shortcoming: Slow during Execution
  • agent收集所有观测给中央，中央得到所有观测才能将决策发给agent
  • Communication and synchronization cost time. 同步，所有都要等待最慢的那个
  • Real-time decision is impossible

Centralized training with decentralized execution:

A central controller is used during training. The controller is disabled after training.

中央控制器指导所有agent的观测 奖励 动作
中央控制器帮助agent训练策略网络
agent有自己的策略网络，根据自己的观测做决策，决策的不需要通信

​

Actor-Critic Method
  • Each agent has its own policy network (actor) $\pi\left(a^{i} \mid o^{i} ; \boldsymbol{\theta}^{i}\right)$
  • The central controller has n value networks (critics) $q\left(\mathbf{o}, \mathbf{a} ; \mathbf{w}^{i}\right)$

Centralized Training:
During training, the central controller knows all the agents’ observations, actions, and rewards

Decentralized Execution
During execution, the central controller and its value networks are not used
